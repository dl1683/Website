My Projects

Parkinson's disease detection. Keywords: Machine learning, Python, Tensorflow, Pytorch, SK-learn, mutlivariate optimization, Downsampling.

This was my most comprehensive project under Dr. Abhyankar. We used the online database as a model to build our own database and run experiments on it. I was responsible for running different directed and undirected models to test out the nature of the data. I also wrote cost functions to determine the more important dimensions of the data.
In the end , we created a method to map the multidimensional input data onto a single binary output set. This gave us a 94% accuracy while not being resource intensive. The algorithm was then formalized as a patent, which has now been monetized. View the terms.
